# To-Do List App (Java)

## Overview
This is a Java-based application that allows users to manage tasks by adding, updating, and deleting items from a to-do list.

## Features
- Add new tasks
- Update existing tasks
- Delete tasks
- Simple task management system

## Tech Stack
- Java

## Project Structure
- src/ contains all source code files

## How to Run
1. Open the project in any Java IDE (IntelliJ, Eclipse)
2. Compile and run the main class

## Purpose
This project was built to practice Java programming, object-oriented concepts, and application logic.
